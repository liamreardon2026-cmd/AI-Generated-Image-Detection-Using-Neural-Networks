Independently developed a deep learning image classification model capable of detecting AI-generated vs. real images using TensorFlow and Convolutional Neural Networks. Trained and evaluated the model on a large dataset containing both authentic and AI-generated images, achieving 76% accuracy on unseen testing data.

Developed and deployed an interactive Streamlit web application that allows users to upload images and receive real-time predictions on whether an image is AI-generated or authentic. The project involved image preprocessing with OpenCV, model tuning to reduce overfitting, and experimentation with different architectures, epochs, and unit sizes to improve performance.

Technologies used include Python, TensorFlow/Keras, OpenCV, NumPy, and Streamlit.
